# Authorization

This repository contains a simple example of authorization implemented using JWT (JSON Web Tokens).

The project is split into client-side and server-side parts to demonstrate how JWT-based authorization can be handled across an application.

## Project Structure

## Overview

- Uses JWT tokens for authorization
- Separates client and server responsibilities
- Intended as a small demo or learning example

## Running the Project

Both the client and server are started using the same command.

```bash
npm run dev
